# MTG-Battlefield
An EDH Utility Tool with Life Totals, Dice Rolls/Coin Flips, and Token Tracker. 
Welcome to MTG Battlefield! This was created initially as the Token Tracker, but quickly became a full utility for an all-in-one MTG tool. 

Home Screen:
This screen gives you your life totals for all players, including tabs for Commander Damage and Player Counters, such as Poison, Rad, Experience, etc. Each player card can be renamed and recolored! You can also use Solo so each player can use their own devices to track life totals and such. This makes remote play easier. New game resets life totals, removes all counters, and removes all tokens within the Token Tracker.

Token Tracker:
This is the meat and potatoes of the app. The tracker allows you to create creature and artifact tokens for the battlefield. The creation screen utilizes Scryfall API to aggregate token card data to auto-generate results when filling the text box, though this is optional and currently being worked on. 
After selecting a token, you can choose quantity, card colors, base Power and Toughness, card type, and abilities (complete with {t}, costs, and custom text). Add to battlefield creates the token. 
Tokens on the battlefield can be tapped and untapped individually, can have P/T adjusted for static abilities and End of Turn abilities (more on this later), and artifact tokens can be animated (Rise and Shine)! Tokens can also be removed individually or all at once via the token's dropdown or via the Clear button at the top of the page, allowing you to clear all tokens, all creature tokens, or all artifact tokens, with a confirmation request. 
When tokens are tapped, an Untap All button will appear at the top. Press this to... untap all tokens...
Certain abilities and cards allow P/T to be modified until the end of turn. These modifiers can be changed via the token's dropdown menu. When you press the End of Turn button (as referenced earlier), all End of Turn modifiers for all tokens will be cleared at once. Pretty neat!

Dice/Coin:
This one is pretty self-explanitory. Press this button, select your dice/coin, peep the result! Select another dice/coin to instantly roll again.
